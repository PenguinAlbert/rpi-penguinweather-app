# 🐧 Penguin Weather

A clean, simple weather dashboard for Raspberry Pi OS.

## 🌦️ Features
- 🌍 Works worldwide (just type "City, Country")
- ⭐ Save favourite locations
- 📅 7-day forecast
- 🖥️ Desktop app (shows in Raspberry Pi menu)
- 🐧 Custom icon included
